# Inductance calculator

RF Inductance Calculator for Single‑Layer Helical Round‑Wire Coils

Original work: https://hamwaves.com/inductance/en/index.html by Serge Y.
Stroobandt, ON4AA

licensed under [GNU GPL version 3](https://www.gnu.org/licenses/gpl.html)

The calculator enabled me to build tuning colils for my short whip antennas and
make contacts from my apartment, even making a few contacts on 160m SSB. I
never thought such a thing was possible. Without ON4AA and others I would have
little chance to build the coils that made this possible.
